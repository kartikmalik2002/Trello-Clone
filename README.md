# Trello-Clone
A project management application
## Description
The Trello is a well known project management application. Here I tried to clone this app.
## Languages Used
1. Kotlin 
2. XML
### IDE 
Android Studio
### Services Used
1. Firebase Authentication
2.  Firebase Firestore
